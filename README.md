# 🧠 SensAI – AI Career Coach

SensAI is an AI-powered career guidance platform designed to help users make smarter career decisions. It provides personalized recommendations, resume guidance, interview preparation, and learning paths based on user goals and skills.

---

## ✨ Features

- 🎯 Personalized career recommendations  
- 📄 Resume building guidance  
- 🎤 Interview preparation tips  
- 📚 Learning path suggestions  
- 💬 Interactive AI-based guidance system  
- 📊 User-focused decision support
