🧠 Sensai-Pro

A powerful full-stack application built with Next.js, designed for scalability, performance, and clean architecture. Sensai-Pro leverages modern tools like Prisma, Tailwind CSS, and custom hooks to deliver a seamless developer and user experience.

🚀 Features
⚡ Built with Next.js (App Router)
🎨 Styled using Tailwind CSS
🧠 Modular and scalable architecture
🔄 Reusable components & hooks
🗄️ Database integration using Prisma
🔐 Middleware support (auth / request handling)
🌱 Environment-based configuration
🏗️ Project Structure
SENSAI-PRO/
│
├── sensai-pro/         # Main project directory
│   ├── app/            # Next.js App Router
│   ├── components/     # Reusable UI components
│   ├── hooks/          # Custom React hooks
│   ├── lib/            # Utility functions & helpers
│   ├── prisma/         # Prisma schema & DB setup
│   │
│   ├── middleware.js   # Middleware (auth, routing, etc.)
│   ├── next.config.js  # Next.js configuration
│   ├── tailwind.config.js
│   ├── postcss.config.js
│   │
│   ├── .env            # Environment variables
│   ├── .env.local      # Local environment overrides
│   │
│   ├── package.json    # Project dependencies & scripts
│   └── README.md       # Project documentation
🧰 Tech Stack
Frontend: Next.js, React
Styling: Tailwind CSS
Backend: Next.js API routes / server components
Database: Prisma ORM
Tooling: ESLint, PostCSS
